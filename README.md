### recreating this project

requirements
- NodeJS
- grunt-cli
- coffeescript
- python 2.7+
- PyPDF2

install dependencies
```
sh ./install
```

collect and parse the pdfs...
```shell
sh ./scrape
```

building vizualization...
```shell
grunt
```
