### recreating this project

requirements
- NodeJS
- grunt-cli
- coffeescript
- python 2.7+
- PyPDF2

install dependencies
```
chmod +x ./install
./install
```

collect and parse the pdfs...
```shell
chmod +x ./scrape
./scrape
```

building vizualization...
```shell
grunt
```
